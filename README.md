# Protoseq Android Studio Project

Android Studio starter project for Protoseq.

Included:
- Gradle project files
- Android manifest
- Compose setup
- Initial theme
- Vertical instrument-style screen
- Base package structure for clock, MIDI, transport, and stochastic modules

Notes:
- Open it in Android Studio and let Gradle sync.
- I could not run a full Android build in this environment, so treat this as a ready-to-import starter project rather than a verified APK build.
