# Protoseq

Protoseq is an open-source MIDI sequencer for Android.

Its first release include aone sequencer  based on the [Turing Machine](https://www.musicthing.co.uk/Turing-Machine/) Eurorack module by Music Thing Modular, bringing that random looping sequencer concept into an Android MIDI environment.

This is the first step in my journey exploring classic and experimental MIDI sequencers for Android.

Protoseq can generate random MIDI note sequences with lock control, allowing patterns to drift, stabilize, or repeat. It can also be used as a modulation source through MIDI CC.

## Capabilities

- Random MIDI note sequence generation
- Lock control for stabilizing or repeating patterns
- MIDI CC output for modulation purposes
- Slew limiter for smoother MIDI CC modulation
- Bernoulli gate
- Sequence length control
- Gate length control
- Random gate length
- Sequence modifier inspired by the Repeater effect in Renoise
